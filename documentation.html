API Documentation

Integrate SolarScheduler with your existing tools and build custom solutions

Authentication

Secure API access using API keys and OAuth 2.0 authentication.

POST /api/v1/auth/token { "api_key": "your_api_key", "secret": "your_secret" }

Jobs API

Create, update, and manage solar installation jobs programmatically.

GET /api/v1/jobs POST /api/v1/jobs PUT /api/v1/jobs/{id} DELETE /api/v1/jobs/{id}

Team API

Manage team members, schedules, and assignments through the API.

GET /api/v1/team/members POST /api/v1/team/schedule GET /api/v1/team/availability

Analytics API

Access business metrics and generate custom reports via API.

GET /api/v1/analytics/revenue GET /api/v1/analytics/jobs GET /api/v1/analytics/team

Webhooks

Receive real-time notifications for job updates and status changes.

POST /api/v1/webhooks { "url": "https://your-app.com/webhook", "events": ["job.completed", "job.started"] }

SDK Libraries

Use our official SDKs for popular programming languages.

# Python pip install solarscheduler-python # JavaScript npm install solarscheduler-js # PHP composer require solarscheduler/php-sdk
